使用PicGo+GitHub搭建博客图床

一、创建图床仓库#

1、创建仓库#

登陆 github ，创建一个公开仓库作为图片存储库。

2、获取个人 token#

此时，需要生成一个token用于PicGo访问图床仓库，访问：settings-tokens ，点击Generate new token。然后设置 token 属性，Expiration：no expiration，Select scopes：repo 全选即可。

3、保存token#

token 生成后只会显示一次，请复制、保存到其他地方以备后续使用。

二、安装PicGo#

可以到 Github 发布页面下载 PicGo 最新安装包。

1、配置图床#

设置PicGo，仓库名的格式是用户名/仓库，一般选择 main 分支，token 是上文中获取的。

  • 存储路径：可以选择让图片单独放在仓库的文件夹中
  • 自定义域名：后面文章再谈使用CDN加速图床，这里可以先空着。

2、PicGo 设置#

如果在粘贴图片时上传失败，可以尝试把内置剪贴板上传功能关闭或开启。

三、安装Obsidian#

去Obsidian 官网下载，并进行安装。

1、安装插件#

点击Obsidian——Preferences——第三方插件——关闭安全模式——社区插件市场。

然后搜索安装以下插件。

2、配置插件#

按以下设置进行配置即可，开启剪贴板自动上传，接口是一一对应的。默认就好，不用修改。

3、使用#

在用写obsidian写文章的时候，往obsidian里吗粘贴图片，就会自动上传到GitHub的图片仓库了。
